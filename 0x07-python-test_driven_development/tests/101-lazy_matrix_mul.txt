=====================================
How to Use 101-lazy_matrix_mul.py
=====================================

This module introduces a matrix multiplication function called ``lazy_matrix_mul(m_a, m_b)``.

Usage
=====

The function ``lazy_matrix_mul(...)`` returns a new matrix representing the multiplication of ``m_a`` by ``m_b``.

::

    >>> lazy_matrix_mul = __import__('101-lazy_matrix_mul').lazy_matrix_mul
    >>> m_a = [
    ...     [1, 2],
    ...     [3, 4],
    ... ]
    >>> m_b = m_a
